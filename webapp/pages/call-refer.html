⚙️ Daily Batch Size

Determines how many doctors appear in each day's call list. The system rotates through all doctors in order and repeats. Incomplete calls carry over to the next day automatically.

ℹ️ How It Works

  • Doctors are ordered by the date they were added.
  • Each day the system picks the next batch_size doctors in sequence.
  • Any calls not marked Done carry over to tomorrow's batch first.
  • Once all doctors are covered the cycle repeats from the top.
  • New doctors join the end of the queue automatically.
  • Each user has an independent rotation offset.
