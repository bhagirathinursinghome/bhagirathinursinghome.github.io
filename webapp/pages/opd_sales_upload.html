🚀 How to deploy the Python normalizer API (free)

  1. Download opd-normalizer-api.zip from the files shared with this page
  2. Go to github.com → New repository → opd-normalizer-api (public)
  3. Upload all files from the ZIP into that repo
  4. Go to railway.app → sign up with GitHub (free)
  5. New Project → Deploy from GitHub repo → select opd-normalizer-api
  6. Wait ~2 min → Railway gives you a URL like https://opd-normalizer-api-production.up.railway.app
  7. Paste that URL into the Normalizer API field above ✅
